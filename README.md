# deforestation-makeathon-2026
Advanced multimodal AI pipeline for the Makeathon Challenge 2026. This project uses Sentinel-1 Radar and Sentinel-2 Optical fusion to detect deforestation and predict the year of occurrence. Key features include Temporal Change Detection ( Δ Δ NDVI), Focal Tversky Loss to minimize False Positives, and 0.5ha geometric filtering for high IoU.
